# Promotion gate

This record is acquisition-only. Do not make it conditional or active until:

- [ ] The 60.75 GB archive is inventoried without extracting untrusted members
      into a grantable directory.
- [ ] A byte-budgeted allowlist identifies separable event, photometry,
      stimulation, and kinematic inputs.
- [ ] Author module names, semantic labels, fitted models, figures, notebooks,
      shuffled controls, and reported summaries are excluded.
- [ ] Module identifiers are deterministically anonymized while preserving
      within-session joins.
- [ ] Animal- and session-grouped intervention holdouts are frozen.
- [ ] Event windows distinguish immediate movement effects from delayed changes
      in module use and sequence variation.
- [ ] Shuffled-lag controls are generated evaluator-side and hidden.
- [ ] `intervention_effects.csv` and `lag_null.csv` receive deterministic schema,
      coverage, ordering, and held-out-direction gates.
- [ ] A pilot demonstrates that neutral extraction, staging, and analysis fit the
      disk, memory, and runtime budget.
- [ ] Leak-guard and blinded-review tests confirm that target semantics and
      conclusions are not encoded in the grant.
