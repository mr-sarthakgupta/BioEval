# Neutral F1-ATPase evidence bundle

`training_observations.csv` separates turnover and coupling-efficiency calibration
data from the nucleotide-binding curves already catalogued from the repository.

`independent_validation_constraints.csv` records observations from predecessor
single-molecule, fluorescence, and structural studies. These constraints are
reserved for model-class validation and must not be converted into fitting targets.

The target paper's Nature Source Data ZIP is not part of this bundle: inspection
shows that it is dominated by fitted parameter ensembles, predictions, simulated
state populations, and figure-organized model outputs. The target Supplementary
Information is also blocked because it states the candidate pruning and conclusions.
