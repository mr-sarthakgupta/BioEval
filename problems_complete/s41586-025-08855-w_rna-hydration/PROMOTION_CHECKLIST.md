# Promotion gate

The bounded four-half-map replication task is active after deterministic density,
matching, leakage, and readiness validation. The remaining items are independent
review and resource-calibration follow-ups before broadening the scope:

- [x] Four half-maps are staged under blinded names with EMDB labels removed
      from MRC headers and exact byte sizes and SHA-256 hashes.
- [x] The RNA reference predates the target footprint and contains only `ATOM`
      polymer records.
- [x] Target solvent models and consensus labels are not acquired or used for
      scoring; full maps, validation products, simulations, code, publication
      files, and result tables are blocked from the evaluated agent.
- [ ] A blinded baseline independently detects candidates per half-map and
      demonstrates deterministic cross-half and cross-reconstruction matching.
- [x] Map-derived fixtures test coordinate bounds, input/parameter hashes, exact
      two-half replication, matching distance, and recomputed density.
- [ ] A blinded baseline calibrates density and local-background thresholds.
- [ ] Map decompression and analysis fit declared disk, memory, and runtime
      budgets in the evaluation container.
- [ ] A structural-biology reviewer confirms that the pre-footprint scaffold
      does not encode target solvent coordinates or target-specific fitting.
