# Promotion gate

The bounded pre-2016 accounting audit is active with direct input recomputation.
The conditions below remain required before adding extrapolated crossing-year or
broader target-reconstruction claims:

- [ ] An independent reviewer confirms that the two source workbooks are
      observations or literature estimates, not fitted target outputs.
- [ ] The pinned commit predates the public article and its MIT license and file
      hashes are recorded.
- [ ] Grantable CSVs contain no repository identity, exact target title, crossing
      year, post-2015 projection, figure value, or target sensitivity result.
- [ ] Biomass carbon-to-dry-mass conversion is specified from an admissible
      pre-cutoff source and tested across a defensible uncertainty range.
- [ ] A blinded solver can recover stable directionality and a broad crossing
      interval without target notebooks, code, figures, or repository search.
- [ ] The artifact evaluator rejects missing definitions, non-finite years,
      reversed intervals, implausible extrapolation, and malformed manifests.
- [ ] The benchmark owner explicitly accepts global material/biomass accounting
      as within the biology-suite boundary.
- [ ] Resource, readiness, grant-boundary, and leak-guard tests pass.
